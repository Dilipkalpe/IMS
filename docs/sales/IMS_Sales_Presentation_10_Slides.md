## Slide 3 — Introducing IMS

**Title:** IMS — integrated operations platform

- Central database — everyone sees live data

**Visual:** High-level diagram (center: **IMS**, around it: Sales | Purchase | Stock | Accounts)

## Slide 4 — Key modules at a glance

**Title:** Everything in one menu  

| Area | What you manage |
|------|-----------------|
| **Sales** | Order → Delivery Challan → Tax Invoice → Return |
| **Purchase** | PO → GRN → Purchase Invoice → Return |
| **Inventory** | Movements, transfers, opening/closing stock |
| **Production** | Work orders, work centers, status |
| **Finance** | Receipts, payments, bank, petty cash |
| **Reports** | Outstanding, due aging, P&L, trial balance |

**Speaker notes:**  
“You can start with Sales + Purchase + Stock, then add Finance and advanced reports when ready.”
